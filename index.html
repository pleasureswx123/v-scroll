长文本滚动示例

虚拟滚动条（Virtual Scrollbar）是一种自定义 UI 组件，用于替代浏览器原生滚动条，提供更一致、更美观的跨平台滚动体验。


本组件 <v-scroll> 基于原生 Web Components 技术实现，使用 Shadow DOM 封装样式，确保组件样式不会污染全局，也不受外部样式影响。


核心特性包括：自定义滚动条轨道与滑块样式、支持鼠标拖拽滑块、点击轨道跳转、上下按钮步进滚动，以及通过 ResizeObserver 自动响应内容高度变化。


组件采用纯函数编程范式，所有业务逻辑均以箭头函数形式定义，禁止使用 class 关键字（customElements.define 的匿名类除外），确保代码的可测试性与可维护性。


样式通过 Vite 自定义插件在构建时将 CSS 文件压缩为 JS 模块，并通过 Import Map 以 $/ 前缀引入，实现了构建层与运行时的解耦。


SVG 图标以 data URI 形式内联在 CSS 中，避免额外的网络请求，提升加载性能。拖拽时使用 grab.svg 作为自定义光标，上下按钮使用 scroll.svg 作为方向指示图标。
